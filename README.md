# lib

|プログラム|用途|
|:----|:----|
|`editdistance.c`|編集距離|
|`readfile.c`|ファイル読み込み|
|`matching.c`|小さい配列が大きい配列のどこに当たるかマッチング|
|`lcs.c`|最長共通部分列<br>このプログラムはパターンがに対応できないので使えない|
|`edata_generator.c`|edata生成|
